# 手工测试步骤 (v1.0.1)

> **前提**:从 `npm run tauri dev` 启动,或安装已发布的 NSIS 包。

## 1. 启动与托盘

| # | 操作 | 期望 |
|---|------|------|
| 1 | 启动应用 | **无主窗口出现**,系统托盘出现 ClipNest 图标 |
| 2 | 右键托盘图标 | 弹出菜单:打开搜索 / 固定收藏 / 最近保存 / 设置 / 退出 |
| 3 | 左键单击托盘图标 | 弹出搜索窗口 |
| 4 | 点击托盘 → 设置 | 弹出设置窗口 |

## 2. 核心闭环:复制 → 保存 → 搜索

| # | 操作 | 期望 |
|---|------|------|
| 5 | 复制 `docker run -d -p 80:80 nginx` | — |
| 6 | 按 **Ctrl+Shift+S** | 弹出保存窗,内容预览显示刚才的 docker 命令 |
| 7 | 标题自动建议为 `docker run` | 标题栏已填,可直接 Enter |
| 8 | 标题改为超过 10 字 | 警告 "标题最多 10 字",保存按钮禁用 |
| 9 | 标题改回 8 字内,按 **Enter** | 显示 "已保存" Toast,窗口 0.3s 后关闭 |
| 10 | 按 **Alt+Space** | 搜索窗打开,搜索框自动聚焦 |
| 11 | 输入 `docker` | 列表显示刚才保存的片段 |
| 12 | 按 **↓** | 选中项下移(虽然只有 1 条,选中无变化) |
| 13 | 按 **Enter** | 复制内容到剪贴板,窗口关闭,系统回到原焦点 |

## 3. 重复保存

| # | 操作 | 期望 |
|---|------|------|
| 14 | 复制同样的 `docker run ...` | — |
| 15 | 按 **Ctrl+Shift+S** | 弹保存窗 |
| 16 | 按 **Enter** 保存 | **不直接保存**,弹出横幅"这条内容已经保存过" |
| 17 | 点击 "取消" | 横幅消失,可继续编辑 |
| 18 | 点击 "仍然保存副本" | 保存成功(用 `save_snippet_force` 跳过检查) |

## 4. AI 标签(需要本地 Ollama)

| # | 操作 | 期望 |
|---|------|------|
| 19 | Ollama 未启动,保存一条 docker 命令 | 正常保存(AI 失败静默回退) |
| 20 | 启动 Ollama,设置 → 启用 AI 标签 → 保存 | 弹窗内出现 "AI 标签" 按钮 |
| 21 | 点击 AI 标签 | 标签数增加,Toast "AI 标签已添加" |

## 5. 搜索键盘操作

| # | 操作 | 期望 |
|---|------|------|
| 22 | Alt+Space 打开搜索 | 焦点在搜索框 |
| 23 | 输入 `docker` + ↓ | 没有匹配项时无响应 |
| 24 | 清空搜索 | 显示最近 50 条 |
| 25 | 选中一条,按 **Enter** | 复制 + 关闭 |
| 26 | 重新打开,选中,按 **Ctrl+C** | 复制,**窗口保持打开**,Toast "已复制" |
| 27 | 按 **Esc** | 窗口关闭 |

## 6. 详情面板

| # | 操作 | 期望 |
|---|------|------|
| 28 | 选中一条记录 | 右侧详情显示标题、类型、标签、完整内容 |

| 36 | 复制一条 URL,选中查看 | 详情面板中 URL 可点击,点击在浏览器打开 |
