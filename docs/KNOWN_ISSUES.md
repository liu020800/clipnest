# Known Issues / 计划中

> v1.0.1 长期稳定版已交付,以下为**已知未解决**或**明确延后**的项目。

## 计划中(后续版本可能恢复)

### 图片保存
- **状态**:DB 字段 `snippets.image_path` 保留,`save_image` Tauri 命令返回明确错误
- **UI**:保存窗不再提供图片入口,设置页的图片捕获 toggle 显示 "计划中" 并禁用
- **影响**:剪贴板读取**仅**支持文本,绝不会读到图片字节流

### AI 摘要持久化
- **状态**:`auto_tag_ai` 返回 `AiTagResult { tags, summary, source }`,前端可使用 `summary` 但**不会**写入数据库
- **原因**:持久化需要 `schema_version = 3` 迁移,需新增 `snippets.summary` 列;在 v1.0.1 范围内避免引入新迁移
- **后续**:若用户需求强烈,可在 v1.0.2 增加一列 `summary TEXT`

## 不做(明确放弃)

- **云同步** — 数据永远在本地
- **账号系统** — 零注册零登录
- **向量数据库** — 全文 FTS5 已足够
- **复杂 AI 知识库** — 仅标签生成,不抽取实体关系
- **多语言 UI** — 仅中文
- **亮色主题** — 仅深色玻璃风
- **WebDAV / 浏览器插件 / 手机端** — 不在 v1.x 路线图中
- **Electron 迁移** — 不会发生

## 已知非关键问题

### 长文本性能
- 50,000 字符以内的内容性能良好;超过会截断
- 如果导入 JSON 含超长片段,搜索时可能略慢(FTS5 仍可工作,但 LIKE 兜底会扫描全文)

### 托盘刷新时机
- 系统托盘菜单中的 "固定收藏" / "最近保存" 在保存后立即刷新;但如果数据库被外部工具直接修改,托盘菜单不会自动同步(需要重启应用)

### 多显示器
- 窗口位置在多显示器之间移动可能丢失(`center()` 在主显示器居中);实际使用中影响不大

### 快捷键冲突
- 如果用户的 Alt+W / Ctrl+Shift+S / Alt+Space 与其他应用冲突,系统优先级由 Windows 决定;ClipNest 暂未提供冲突检测提示
